# Ecommerce Payment Gateway Testing Project – OpenCart

This project demonstrates QA testing of an ecommerce payment flow using the OpenCart demo website.

Website Tested:
https://demo.opencart.com/

Project simulates real ecommerce platforms like Amazon and Flipkart.

## Testing Types
- Manual Testing
- Payment Gateway Testing
- API Testing using Postman
- Database Validation using SQL

## Tools Used
- Excel
- SQL
- Browser DevTools

## Test Coverage
- Product Search
- Product Page Validation
- Add to Cart
- Cart Quantity Validation
- Checkout Process
- Payment Flow

## Bugs Identified
- Cart accepts extremely large quantity
- Search works with empty input
- Cart allows invalid quantity
